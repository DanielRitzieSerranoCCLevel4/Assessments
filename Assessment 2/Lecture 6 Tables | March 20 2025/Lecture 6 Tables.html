    Accessibility and Tables
    Use table header elements ([th] tags) to indicate column or row headings.
    Use the caption element to provide a text title or caption for the table.
    Complex tables: Associate table cell values with their corresponding headers
    [th] tag id attribute
    [td] tag headers attribute




    Table Row Groups

    1. [thead] (Table header rows)
    -Contains column headers (Product, Quantity, Price).
    -Styled with a gray background for distinction.
    2. [tbody] (Table body rows)
    -Holds the main data (list of products and their quantities & prices).
    3. [tfoot] (Table footer rows)
    -Used for summary information, such as the total revenue.
    -Styled with a gray background like the [thead].

    ✨ Benefits of Using [thead], [tbody], and [tfoot]
    ✔ Improves readability and makes tables easier to manage.
    ✔ Enhances accessibility for screen readers.
    ✔ Enables better styling using CSS.
    ✔ Useful for long tables (e.g., reports, invoices, or financial statements).
