HTML5 Video Example




We can make the audio autoplay with the following code:
[audio controls autoplay]
[source src=“rain.mp3" type="audio/mpeg"]



We can make the audio loop with the following code:
[audio controls loop]
[source src=“rain.mp3" type="audio/mpeg"] [/audio]

We can add video in HTML through video tag:


Once the video is downloaded save in desktop or copy in your visual studio project/same project folder.
Try the following code:
Use the opening and closing tag for video and type controls
Use the source tag followed by src and insert the complete path and extension of the video
Write the type and select video/mp4 [video width="320" height="240" controls ] [source src="HTML.mp4" type="video/mp4"] [/video]




Adding the last code will make the videos to be autoplayed and set on mute
autoplay=1 → Enables autoplay when the page loads.
mute=1 → Mutes the video (required for autoplay to work on most browsers).



Adding the last code will make the videos to be autoplayed, set on mute with repeat
autoplay=1 → starts playing automatically
mute=1 → Mutes the video (required for autoplay to work on most browsers).
Loop=1 → Enables looping
